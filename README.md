# AD&D Combat Tools

Tools for running AD&D 1e sessions in the browser.

- AD&D Combat Calculator: `/calculator`
- AD&D Battle Grid: `/battle`
- AD&D Random Dungeon Generator: `/dungeon`

Local development

- `npm run dev` then open `http://localhost:3000/dungeon` to try the Dungeon Generator (or `/battle`, `/calculator`).

Production

- Published via GitHub Pages at `https://tunesmith.github.io/adnd-combat-tools` (use the `basePath` in links).
